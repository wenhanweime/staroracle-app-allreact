# 快速开始指南

## 🚀 立即运行（3 步）

### 1. 打开项目
```bash
cd pixelplanet_final
open PixelPlanetsSwiftUIApp/PixelPlanetsSwiftUIApp.xcodeproj
```

### 2. 在 Xcode 中
- 选择 Scheme: `PixelPlanetsSwiftUIApp`
- 选择 Destination: `iPhone 15 Pro` (或任意 iOS 模拟器)

### 3. 运行
- 按 `⌘R` 或点击运行按钮

## ⚠️ 如果遇到问题

如果 Xcode 提示找不到 `PixelPlanetsCore` 模块，请按照 `XCODE_SETUP.md` 中的详细步骤重新链接 Swift Package。
